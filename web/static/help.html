1. 画面の見方（全体構成）

bunri DAW の画面は大きく5つのエリアで構成されています。

ヘッダー（BPM・再生/停止・保存/書出）
左パネル
(シンセ/ドラム
/FX/ファイル)
タイムライン（トラック＋クリップ）
ピアノロール（ノート打ち込み）
オートメーション
エリア役割
ヘッダー（上部）BPM（テンポ）設定、再生/停止/録音ボタン、プロジェクト保存/書出
左パネル4つのタブ（シンセ・ドラム・FX・ファイル）を切り替えて操作する設定エリア
タイムライン（中央上）音声の「クリップ」を横方向に並べるメインの作業エリア。複数のトラック（横の列）がある
ピアノロール（中央下）シンセ用のメロディを音符（ノート）単位で打ち込むエリア
オートメーション（最下部）音量など時間的に変化させたいパラメータを制御するカーブエリア
ヒント：画面上のボタンやスライダーにマウスを乗せると、画面最下部のステータスバーに説明が表示されます。

2. クイックスタート：最初の1曲を作る

まず何をすればいいのかわからない方向けに、最短手順で音を鳴らす方法を説明します。

方法A：シンセでメロディを作る（おすすめ）

  1. 楽器を選ぶ
    左パネルの「シンセ」タブを開きます（初期状態で開いています）。
    「楽器（GM音源）」のドロップダウンから好きな楽器を選びます。
    例：「ピアノ」「バイオリン」「ナイロンギター」「コーラス Aah」など。
  2. ピアノロールにノートを置く
    画面中央下の「ピアノロール」エリアを見てください。
    左端に鍵盤（C2〜B6）が縦に並び、右側にマス目が広がっています。
    マス目の上でダブルクリックすると、青い四角（＝ノート＝音符）が1つ置かれます。

    ノートの位置の意味：
    ・縦の位置 ＝ 音の高さ（上に行くほど高い音。左の鍵盤を参考にしてください）
    ・横の位置 ＝ 音が鳴るタイミング（左が曲の始まり、右に行くほど後ろ）

    例えば「ドレミ」を打ちたいなら：
    ・C4の行の左端あたりでダブルクリック（ド）
    ・D4の行のすぐ右隣でダブルクリック（レ）
    ・E4の行のさらに右隣でダブルクリック（ミ）
  3. 追加先トラックを確認する
    左パネル下部の「追加先トラック」で、音声を入れたいトラックを選びます。
    初期状態では「Track 1」「Track 2」があります。
  4. 「シーケンスをレンダリング」ボタンを押す
    左パネル最下部の青いボタンを押すと、ピアノロールに置いたノートが実際の音声に変換され、選んだトラックに自動追加されます。
    タイムライン上に青い帯（＝クリップ）が表示されれば成功です。
  5. 再生する
    画面上部の ▶（再生） ボタンを押すと、タイムラインに並んだ全クリップが再生されます。
    ■（停止） で止まります。

方法B：ドラムを追加する

  1. 左パネル上部のタブから 「ドラム」 をクリック。
  2. 「パターン」で好きなリズムを選ぶ（「8ビート」がおすすめ）。
  3. 「小節数」はそのまま4でOK。
  4. 「ドラム生成 → トラックに追加」 ボタンを押す。
  5. タイムラインに「Drum」トラックが自動作成され、クリップが追加されます。
  6. ▶ で再生して確認。

方法C：手持ちのWAVファイルを使う

  1. 左パネル上部のタブから 「ファイル」 をクリック。
  2. 「ファイルを選択」からPCのWAVファイルを選びます。
  3. 「追加先トラック」で入れたいトラックを選ぶ。
  4. 「選択したトラックに追加」 ボタンを押す。
ヒント：WAVファイルはタイムラインに直接ドラッグ&ドロップしても追加できます。

組み合わせて曲にする

上のA・B・Cを組み合わせて、たとえば：
・Track 1 にピアノのメロディ（方法A）
・Track 2（Drum）にドラム（方法B）
・Track 3 にボーカルのWAVファイル（方法C）

…と複数トラックに音を配置すれば、それが同時に再生されて「曲」になります。

3. ヘッダーとトランスポート

画面最上部の横長エリアです。

要素説明
BPM 曲のテンポ（速さ）。数値が大きいほど速い曲になります。
・60 = 1秒に1拍（ゆっくり）
・120 = 1秒に2拍（標準的なポップス）
・140 = 速めのダンスミュージック
シンセやドラムの生成はこのBPMを基準に行われます。
拍子 1小節に何拍入るか。
・4/4 = 1小節4拍（ほとんどの曲はこれ）
・3/4 = ワルツ
・6/8 = マーチ風
▶（再生）タイムラインの全トラックを先頭から同時に再生します。
■（停止）再生を止めて先頭に戻ります。
●（録音）PCのマイクから録音を開始します。同時に再生も走るので、曲を聴きながら歌やギターを録音できます。もう一度押すと録音が止まり、Recトラックに追加されます。
メトロノーム再生中に「カチカチ」とBPMに合わせたクリック音を鳴らします。リズムを取りやすくなります。もう一度押すとOFF。
時間表示現在の再生位置を「分:秒」で表示します。
保存現在のプロジェクト状態（トラック構成、ノート、オートメーション）をサーバーに保存します。
読込最後に保存したプロジェクトを読み込みます。（注：音声データ自体はプロジェクトに含まれないため、WAVファイルは再インポートが必要です）
書出全トラックをミックスした1つのWAVファイルをダウンロードします（完成した曲のファイル）。
❓ウェルカムガイド（簡易説明）を再表示します。

4. タイムライン（トラックとクリップ）

画面中央上部の最も大きなエリアです。ここが曲の全体像を管理する場所です。

「トラック」とは？

トラックは横に伸びる1列のレーンです。楽器ごと、パートごとに1トラック使います。
例：メロディ用トラック、ドラム用トラック、ボーカル用トラック…

トラック構成のイメージ： Track 1 [ピアノ] ┃▓▓▓▓▓▓▓▓▓░░░░░░░░░░▓▓▓▓▓▓░░░░░ Track 2 [Drum] ┃▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ Track 3 [Vocal] ┃░░░░░░░▓▓▓▓▓▓▓▓▓▓▓░░░░░░░░░░░░░ ┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 0秒 → 時間 ▓ = クリップ（音声の塊） ░ = 無音

トラックの操作

操作やり方
トラック追加タイムライン下部の「+ トラック追加」ボタンをクリック
ミュート（M）トラック左端の「M」ボタン。押すとそのトラックだけ音が消えます（赤くなる）。もう一度押すと解除
ソロ（S）トラック左端の「S」ボタン。押すとそのトラックだけが聞こえます（黄色くなる）。他のトラックを一時的にミュートして、特定パートだけ確認したいときに使います
トラック削除トラック左端の「×」ボタン

「クリップ」とは？

クリップはタイムライン上に表示される青い帯で、1つの音声データを表します。
シンセでレンダリングした音声、ドラム、読み込んだWAVファイルはすべてクリップとしてタイムラインに追加されます。

クリップの操作

操作やり方
移動クリップをマウスでドラッグすると横方向に移動できます。再生開始位置を調整できます
削除クリップの上で右クリック
WAV追加（D&D）PCのエクスプローラからWAVファイルをタイムラインに直接ドラッグ&ドロップ
ヒント：クリップの横方向の位置 = 曲の中で「いつ」その音が鳴るか、です。
ドラムを最初から最後まで鳴らし、メロディは少し後から入る…といった配置がドラッグで可能です。

5. シンセパネル（音を作る）

左パネル「シンセ」タブの詳細です。ここで楽器の音色を決め、ピアノロールのノートを音声に変換します。

楽器（GM音源）

最も重要な設定です。ドロップダウンに100種類以上の楽器がカテゴリ別に並んでいます。
選んだ楽器のリアルなサンプル音で音声が生成されます。

カテゴリ代表的な楽器
ピアノピアノ、ブライトピアノ、エレクトリックピアノ、ハープシコード
ギターナイロンギター、スチールギター、ジャズギター、ディストーションギター
ベースアコースティックベース、フィンガーベース、シンセベース
ストリングスバイオリン、ビオラ、チェロ、ストリングアンサンブル
コーラス/ボイスコーラス Aah、コーラス Ooh、シンセボイス
ブラストランペット、トロンボーン、フレンチホルン、ブラスセクション
木管フルート、クラリネット、オーボエ、ピッコロ、リコーダー
シンセシンセリード、シンセパッド各種

「カスタム波形（下で選択）」を選ぶと、GM音源の代わりに簡易合成エンジンが使われます。「カスタム波形設定」を開くと、Sine/Square/Sawtoothなどの基本波形や簡易楽器（ギター、バイオリン等）を選べます。音質はGM音源のほうが格段に上なので、基本的にはGM音源を使うことをおすすめします。

ADSR（エンベロープ）

GM音源使用時は通常変更不要ですが、カスタム波形使用時に音の鳴り方を調整できます。

音量の時間変化（ADSR）： 音量 ↑ 1.0 ┃ /\ ┃ / \___________ 0.6 ┃ / | Sustain \ ┃ / | \ 0.0 ┃/ | \____ ┗━━━━━━━━━━━━━━━━━━━━━→ 時間 A D S R A(Attack) = 音が立ち上がるまでの時間 D(Decay) = 最大からSustainレベルまで下がる時間 S(Sustain) = 鍵盤を押し続けている間の音量 R(Release) = 鍵盤を離してから消えるまでの時間

追加先トラック

レンダリング結果をどのトラックに追加するか選びます。

「シーケンスをレンダリング」ボタン

これが最も重要なボタンです。ピアノロールに置いたノートを、選んだ楽器の音色で実際の音声に変換し、指定したトラックに追加します。
ピアノロールにノートが1つもない状態では押しても何も起きません。

6. ピアノロール（メロディを打ち込む）

画面中央下のマス目エリアです。シンセで音声化するメロディや和音を「ノート」として打ち込みます。

ピアノロールのイメージ： B4 ┃░░░░░░░░░░░░░░░░░░░░░░░░ A4 ┃░░░░░░░░░░░░░░░░░░░░░░░░ G4 ┃░░░░░░░░░░░░░░▓▓▓▓░░░░░░ ← ノート（G4, ソ） F4 ┃░░░░░░░░░░░░░░░░░░░░░░░░ E4 ┃░░░░░░░░▓▓▓▓░░░░░░░░░░░░ ← ノート（E4, ミ） D4 ┃░░░░▓▓▓▓░░░░░░░░░░░░░░░░ ← ノート（D4, レ） C4 ┃▓▓▓▓░░░░░░░░░░░░░░░░░░░░ ← ノート（C4, ド） ┗━┿━━━┿━━━┿━━━┿━━━┿━━━┿━━→ 時間 拍1 拍2 拍3 拍4 拍5 拍6 ▓ = ノート ░ = 空き

操作方法

操作やり方
ノートを追加マス目の上でダブルクリック。クリックした位置の音階・タイミングにノートが配置されます
ノートを移動ノート（青い四角）をマウスでドラッグ。音の高さやタイミングを変更できます
ノートの長さを変えるノートの右端をドラッグして横に伸縮します
ノートを選択ノートをクリック。選択されたノートは赤くなります
ノートを削除ノートを選択した状態でDeleteキー

7. ドラムパネル

左パネル「ドラム」タブの詳細です。プリセットパターンからドラムトラックを自動生成します。

パターン

パターン特徴使うジャンル
8ビート最も基本的なリズム。キックが1拍目と3拍目、スネアが2拍目と4拍目ポップス、ロック全般
4つ打ちキックが全拍で「ドンドンドンドン」テクノ、ハウス、EDM
ボサノバシンコペーションのある穏やかなリズムボサノバ、ラウンジ
レゲエスネアが裏拍に入る独特のリズムレゲエ、ダブ

設定項目

項目説明
小節数ドラムパターンを何小節分繰り返すか。4小節が1フレーズの一般的な長さ。曲全体をカバーしたい場合は小節数を増やしてください
音量ドラムの音量（0〜1）。他のパートとのバランスを調整
追加先トラック「新規作成（Drum）」で専用トラックが自動作成されます。既存トラックを選ぶこともできます
ヒント：ドラムのBPMは画面上部のBPM設定が使われます。曲全体のテンポを先に決めてからドラムを生成すると、他のパートとぴったり合います。

8. FXパネル（エフェクト）

左パネル「FX」タブの詳細です。トラック上の音声にエフェクト（音響処理）をかけます。

注意：現在のエフェクト機能はクリップ単位でサーバー側処理を行うもので、トラック上にクリップが必要です。

エフェクト一覧

エフェクト何をするかどんなときに使う？
EQ (3バンド)低音・中音・高音の音量バランスを調整こもった音をクリアにしたい、低音を強調したい等
コンプレッサー音量の大小差を均一にするボーカルの音量ムラを整えたい、音圧を上げたい等
リバーブ残響音を加えるホールで録ったような広がりを出したい
ディレイやまびこのように音を繰り返す空間的な広がりやリズミカルな効果
ノーマライズ音量を最大まで自動調整録音が小さすぎるとき
ピッチシフト音の高さを変える（速度は変わらない）キーを変えたい、ハモリパートを作りたい
タイムストレッチ再生速度を変える（音の高さは変わらない）テンポを合わせたい

使い方

  1. 「対象トラック」で処理したいトラックを選ぶ
  2. 「エフェクト」ドロップダウンでエフェクトを選ぶ
  3. 表示されるパラメータのスライダーを調整する
  4. 「エフェクト適用」ボタンを押す

9. ファイルパネル（読み込み・録音）

左パネル「ファイル」タブの詳細です。

WAVファイルの読み込み

  1. 「ファイルを選択」ボタンをクリックし、PCからWAVファイルを選びます（複数選択可）
  2. 「追加先トラック」で配置するトラックを選びます
  3. 「選択したトラックに追加」ボタンを押すと、トラックのタイムラインにクリップとして追加されます
ヒント：WAVファイルはタイムラインに直接ドラッグ&ドロップでも追加できます。こちらのほうが直感的かもしれません。

マイク録音

  1. 「マイク録音開始」ボタンを押します。ブラウザがマイクの使用許可を求めますので「許可」を選んでください
  2. マイクに向かって声や楽器を録音します
  3. 「録音停止」ボタンを押すと、録音された音声が「Rec」トラックに自動追加されます
ヒント：ヘッダーの●（録音）ボタンを使うと、既存トラックを再生しながら同時に録音できます。カラオケのように伴奏を聴きながら歌を録音したい場合に便利です。

10. オートメーション

画面最下部のカーブエリアです。音量などのパラメータを時間的に変化させることができます。

オートメーションのイメージ： 100% ┃ ●━━━━━━● ┃ ╱ ╲ ┃ ╱ ╲ 50% ┃ ●━━● ╲ ┃ ●━━━ 0% ┃ ┗━━━━━━━━━━━━━━━━━━━━━━━→ 時間 曲の頭 サビ エンディング ● = コントロールポイント（ドラッグで移動可能）

例えば「サビで音量を上げて、エンディングでフェードアウトする」といった表現に使います。

操作方法

操作やり方
ポイント追加カーブエリア上でダブルクリック
ポイント移動ポイントをドラッグ
ポイント削除ポイントの上で右クリック

11. 保存・書き出し

プロジェクト保存（「保存」ボタン）

現在の作業状態（トラック構成、ノート配置、オートメーション）をサーバーに保存します。
後日「読込」ボタンで復元できます。

注意：音声データ（WAVファイル）自体はプロジェクトに含まれません。読み込み後、WAVファイルは再度インポートする必要があります。シンセやドラムで生成した音声も同様に再レンダリングが必要です。

WAV書き出し（「書出」ボタン）

これが最終成果物を作るボタンです。
タイムライン上の全トラック（ミュートされていないもの）をミックスし、1つのWAVファイルとしてダウンロードします。
友達に送ったり、音楽配信サイトにアップロードできる完成ファイルです。

  1. タイムラインに全パートを配置し終えたら、ミュート/ソロが意図通りか確認する
  2. ▶で最終確認の再生をする
  3. 「書出」ボタンを押す
  4. 少し待つと「bunri_mix.wav」がダウンロードされます

12. 用語集

用語意味
